# Audio Assets for Video Studio

This directory contains royalty-free audio assets for PersonaForge Video Studio.

## Directory Structure

```
audio/
├── music/          # Background music tracks by mood
├── sfx/            # Sound effects for impact moments
├── ambience/       # Ambient soundscapes
└── README.md       # This file
```

## Required Files

### Music (`/music/`)

Background music organized by mood. Each file should be:
- Format: MP3 or WAV
- Duration: 60-180 seconds (loops seamlessly)
- Volume: Normalized to -14 LUFS

### SFX (`/sfx/`)

Sound effects for impact moments. Each file should be:
- Format: MP3 or WAV
- Duration: 0.5-3 seconds
- Volume: Normalized

| Type | Description | Filename |
|------|-------------|----------|
| Whoosh | Quick transition | `whoosh.mp3` |
| Swoosh | Softer transition | `swoosh.mp3` |
| Impact | Deep hit | `impact-deep.mp3` |
| Impact Soft | Gentle accent | `impact-soft.mp3` |
| Boom | Dramatic boom | `boom.mp3` |
| Rise | Building tension | `riser.mp3` |
| Drop | After climax | `drop.mp3` |
| Sparkle | Magical accent | `sparkle.mp3` |
| Chime | Bell-like accent | `chime.mp3` |
| Ding | Notification tone | `ding.mp3` |
| Thunder | Distant thunder | `thunder-distant.mp3` |
| Heartbeat | Dramatic pulse | `heartbeat.mp3` |
| Breath | Deep breath | `breath-deep.mp3` |

### Ambience (`/ambience/`)

Background atmosphere. Each file should be:
- Format: MP3 or WAV
- Duration: 60+ seconds (seamless loop)
- Volume: Very quiet (will be mixed at ~8%)

| Type | Description | Filename |
|------|-------------|----------|
| Room | Subtle room tone | `room-tone.mp3` |
| Outdoor | Birds, nature | `outdoor-birds.mp3` |
| Rain | Gentle rain | `rain-gentle.mp3` |
| Wind | Light wind | `wind-light.mp3` |
| Crowd | Murmuring crowd | `crowd-murmur.mp3` |
| Space | Sci-fi ambient | `space-ambient.mp3` |
| Fire | Crackling fire | `fire-crackle.mp3` |
| Ocean | Ocean waves | `ocean-waves.mp3` |
